## 👨‍💻 How to Setup

```bash
  # Clone the project
  $ git clone https://github.com/Cafety1/PRojetoFinalFinal.git
```
```bash
  # Enter directory
  $ cd cafeinaprojeto
```

```bash
  # Install the dependencies, if you use npm
  $ npm install
```
<p align="center">or</p>

```bash
  # Install the dependencies, if you use yarn
  $ yarn
```



```bash
npm run test:unit
```
